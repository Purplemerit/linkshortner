# URL Shortener - Privacy-First Link Management Platform

## PROJECT OVERVIEW

This is a comprehensive Privacy-First URL Shortener application built with Next.js 14, TypeScript, and React 18. The application provides 50+ MVP features including link shortening, custom domains, QR code generation, real-time analytics, team collaboration, and a Chrome extension.

**Key Differentiators:**
- Privacy-first analytics (no cookies, IP anonymized)
- Broken link monitoring
- Most features at lowest price ($9-15/mo)
- Beautiful minimalist UI (white background, purple #9333ea accents)
- Self-hosted + managed options

## TECH STACK

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **UI Library:** React 18
- **Styling:** TailwindCSS
- **Charts:** Recharts
- **QR Codes:** qrcode.react
- **Package Manager:** npm

## API ENDPOINTS

All endpoints are mock implementations using Next.js API routes:

- `POST /api/links` - Create new short link
- `GET /api/links` - List all links
- `GET /api/links/check-availability?code=xxx` - Check custom code availability
- `GET /api/links/[id]/analytics` - Get link analytics
- `POST /api/links/[id]/tags` - Update link tags
- `POST /api/domains/verify` - Verify custom domain
- `GET /api/qr/[id]` - Generate QR code
- `POST /api/team/members` - Invite team member
- `DELETE /api/team/members/[memberId]` - Remove team member

## COMMON TASKS

**Add a new component:**
1. Create file in `src/components/ComponentName.tsx`
2. Use 'use client' directive
3. Define TypeScript interfaces
4. Use TailwindCSS with purple theme
5. Add error handling
6. Make responsive

**Add a new API endpoint:**
1. Create route file in `src/app/api/endpoint/route.ts`
2. Export async function (GET, POST, etc.)
3. Return NextResponse.json()
4. Use dummy data from `src/lib/dummy-data.ts`

**Add a new page:**
1. Create `src/app/pagename/page.tsx`
2. Add metadata export
3. Use components from `src/components/`
4. Follow design system
